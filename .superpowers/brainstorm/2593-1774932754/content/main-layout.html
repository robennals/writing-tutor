If I was writing about my cat, instead of "my cat is fluffy", I might say "my cat has long orange fur that sticks up like a lion's mane." See how the details help you picture it?

Key Design Decisions

  • Side-by-side layout - Editor left, tutor right. Owen can write and see guidance simultaneously.
  • Step progress bar - Shows where Owen is in the writing process (topic → brainstorm → organize → draft → review → revise → done)
  • Quick action buttons - "I'm done writing", "I need help", "What's next?" so Owen doesn't always have to type
  • Tips in green boxes - Examples on different topics so the tutor never writes Owen's essay for him
  • Warm dark theme - Easier on the eyes, cream-on-dark-gray (not pure black/white) for dyslexia friendliness
  • Word count - Gentle metric, not a target
  • Text-to-speech buttons - On both the essay and tutor messages

Note: The actual app will use a proper dyslexia-friendly font with generous line spacing (1.8+). Colors shown here are approximate - the final theme will use warm grays and cream tones to reduce eye strain.
